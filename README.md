# 🎮 RayMC Manager
**The Ultimate Local Minecraft Server Manager**

![Banner Placeholder](https://via.placeholder.com/800x200/121212/00ffcc?text=RayMC+Manager+Banner)

Welcome to the official public tracker for **RayMC Manager**, proudly created by **Ray**. This tool allows you to effortlessly run, manage, and configure local Minecraft servers straight from your PC using your system's resources.

### ✨ Features
* 🚀 **1-Click Installs:** Automatically download server cores (Paper, Forge, Fabric).
* 🖥️ **Live Console:** Hidden CMD backend with a sleek frontend real-time terminal.
* 🧩 **Modrinth Integration:** Direct plugin and mod downloads to your server folders.
* 🌍 **Public Tunnels:** Easy integration with Playit and temporary SSH tunnels (Pinggy).
* 🎨 **Modern UI:** Deep dark mode, Minecraft aesthetics, and custom background support.
